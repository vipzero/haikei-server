### Event 作成

```
$ vi .envrc EVENT_ID
EVENT_ID=xxxxxx
$ yarn setup			 		# EVENT_ID から動作
$ yarn setup:anison
```

### 終了

アーカイブする。
イベント中の履歴は Firestore からロードするが、アーカイブ後は履歴を Cloud Storage からダウンロードするようになる。

```
yarn close:archive {event_ia}
```
